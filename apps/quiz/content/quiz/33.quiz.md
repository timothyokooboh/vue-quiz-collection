::question{:type="questionType"}
Which Vue router component is used to render the matched component for the current route?

::

::explanation{:type="explanationType"}

**Explanation: RouterView is a built-in component provided by Vue router for rendering the matched component for the current route.**

::

::vueTip{:type="tip" :reference="https://router.vuejs.org/api/variables/RouterView.html" :position="33"}

#text

Use RouterView to render the component that matches the current route in your Vue Router application.

#code

```js
<RouterView />
```
