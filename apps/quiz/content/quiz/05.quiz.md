::question{:type="questionType"}
Which Vue.js helper function is used to generate unique per-application IDs for accessibility attributes?

::

::explanation{:type="explanationType"}

**Explanation: useId can be used to generate IDs for form elements and accessibility attributes.**

<br />

IDs generated by useId() are unique-per-application. Multiple calls in the same component will generate different IDs; multiple instances of the same component calling useId() will also have different IDs.

<br />

```js
<script setup>
import { useId } from 'vue'

const id = useId()
</script>

<template>
  <form>
    <label :for="id">Name:</label>
    <input :id="id" type="text" />
  </form>
</template>
```
