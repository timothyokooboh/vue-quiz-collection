::question{:type="questionType"}
Which of the following is not true about pinia plugins?

::

::explanation{:type="explanationType"}

**State change inside plugins can not trigger store subscriptions.**

<br />

State changes inside a plugin happen before the store is active and therefore do not trigger any subscriptions.


::

::vueTip{:type="tip" :reference="https://pinia.vuejs.org/core-concepts/plugins.html#Adding-new-external-properties" :position="53"}

#text

When adding external properties or class instances that come from other libraries, they should be wrapped with markRaw()

#code

```js
import { markRaw } from 'vue'
import { router } from './router'

pinia.use(({ store }) => {
  store.router = markRaw(router)
})
```
