::question{:type="questionType"}
Which of the following is not a feature of RouterLink?

::

::explanation{:type="explanationType"}

**Explanation: RouterLink does not prevent navigation to invalid routes.**

<br />

- RouterLink changes the URL without reloading the page.
- RouterLink handles URL generation from the declarative object prop "to", which contains properties like `name`, `path`, `params`, and `query`.
- RouterLink encodes URL. This means it automatically converts characters in the URL into a format that can be transmitted over the Internet.

::
