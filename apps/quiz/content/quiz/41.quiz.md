::question{:type="questionType"}
In the code snippet below, when will code in the if block be executed?

<br />

```js
const navigationResult = await router.push('/my-profile')

if (navigationResult) {

} else {

}
```

::

::explanation{:type="explanationType"}

**Explanation: The if block will be executed when navigation is prevented and user stays on the same page.**

<br />

The code snippet provides a way to check if navigation was prevented.

<br />

```js
const navigationResult = await router.push('/my-profile')

if (navigationResult) {
  // navigation prevented
} else {
  // navigation succeeded
}
```
