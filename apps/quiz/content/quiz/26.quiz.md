::question{:type="questionType"}
When using script setup, which compiler macro can be used to expose properties and methods to parent components?

::

::explanation{:type="explanationType"}

**Explanation: The defineExpose macro can be used to expose properties and methods to parent components.**

<br />

```js
<script setup>
const count = ref(0);

defineExpose({
  count
});
</script>
```
