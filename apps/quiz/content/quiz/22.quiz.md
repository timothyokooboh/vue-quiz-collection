::question{:type="questionType"}
Which built-in component allows us to teleport a part of a component's template into a DOM node that exists outside the DOM hierarchy of that component.

::

::explanation{:type="explanationType"}

**The answer is Teleport.**

<br />

In this example, the modal content is teleported to the document body.

<br />

```js
<Teleport to="body">
  <div v-if="open" class="modal">
    <p>Hello from the modal!</p>
    <button @click="open = false">Close</button>
  </div>
</Teleport>
```

::
