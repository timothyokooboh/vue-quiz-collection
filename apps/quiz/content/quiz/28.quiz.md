::question{:type="questionType"}
In the code snippet below, how can the message state be exposed to the slot so that it can be used within the slot content of the parent component?

<br />

```js
const message = ref('Hello, World!');

<div>
  <slot></slot>
</div>

```

::

::explanation{:type="explanationType"}

**Explanation: This is the concept of scoped slots, where data from the child component can be passed to the slot content in the parent component.**
