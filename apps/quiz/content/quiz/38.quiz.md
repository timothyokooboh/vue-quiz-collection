::question{:type="questionType"}
Which of the following is not a benefit of navigating with name instead of path?

::

::explanation{:type="explanationType"}

**Explanation: Navigating with name has many advantages but it does not prevents navigation to invalid routes.**

<br />

Navigating with name has the following advantages:

- No hardcoded URLs.
- Avoids URL typos.
- Automatic encoding of params.
::

::vueTip{:type="tip" :reference="https://router.vuejs.org/guide/essentials/named-routes.html" :position="38"}

#text

Use named routes for navigation to avoid hardcoded URLs, prevent typos, and enable automatic parameter encoding.

#code

```js
router.push({ name: 'User', params: { id: 5 } })
```
