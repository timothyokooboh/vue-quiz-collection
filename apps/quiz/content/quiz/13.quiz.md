::question{:type="questionType"}
Which of the following is not a built-in v-model modifier?

::

::explanation{:type="explanationType"}

**Explanation: capitalize is not a built-in v-model modifier.**

::

::vueTip{:type="tip" :reference="https://vuejs.org/guide/essentials/forms.html#modifiers" :position="13"}

#text

Vue provides built-in v-model modifiers like lazy, number, and trim for common input transformations.

#code

```js
<input v-model.lazy="msg" />
<input v-model.number="age" />
<input v-model.trim="name" />
```
