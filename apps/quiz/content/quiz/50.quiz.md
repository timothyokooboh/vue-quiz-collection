::question{:type="questionType"}
Which Pinia method lets you change multiple states at once?

<br />

```js
import { useCounterStore } from '@/stores/counter'

const store = useCounterStore()
const { name, doubleCount } = store
```

::

::explanation{:type="explanationType"}

**Explanation: $patch lets you change multiple states at once.**

<br />

```js
<script setup>
store.$patch({
  count: store.count + 1,
  age: 120,
})
</script>
```

::
