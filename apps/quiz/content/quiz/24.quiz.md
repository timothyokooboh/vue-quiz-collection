::question{:type="questionType"}
Which of the following can help to improve page load performance in a Vue.js application?

::

::explanation{:type="explanationType"}

**Explanation: Code splitting is where a build tool splits the application bundle into multiple smaller chunks.**

<br />

With proper code splitting, features required at page load can be downloaded immediately, with additional chunks being lazy loaded only when needed, thus improving performance.

::
