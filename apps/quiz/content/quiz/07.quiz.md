::explanation{:type="explanationType"}

**Components with multiple root nodes do not have an automatic attribute fallthrough behavior. If $attrs are not bound explicitly, a runtime warning will be issued.**

<br />

The warning will be suppressed if $attrs is explicitly bound:

<br />

```js
<template>
  <header></header>
  <main v-bind="$attrs"></main>
  <footer></footer>
</template>
```

::

::vueTip{:type="tip" :reference="https://vuejs.org/guide/components/attrs.html#attribute-inheritance-on-multiple-root-nodes" :position="7"}

#text

For components with multiple root nodes, explicitly bind $attrs to one element using v-bind="$attrs" to enable attribute fallthrough and avoid warnings.

#code

```js
<template>
  <header></header>
  <main v-bind="$attrs"></main>
  <footer></footer>
</template>
```
