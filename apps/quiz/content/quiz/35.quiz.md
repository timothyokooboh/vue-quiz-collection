::question{:type="questionType"}
Which of the following is the correct way of registering a route with a dynamic segment?

::

::explanation{:type="explanationType"}

**Explanation: { path: '/posts/:id', component: Post } is the correct way of registering a route with a dynamic segment.**

<br />
When a route is matched, the value of its params will be exposed as route.params.

```js
<template>
  <div>
    Post: {{ $route.params.id }}
  </div>
</template>
```
::
