::question{:type="questionType"}
The code below shows how to implement two-way data binding, thereby allowing a component to accept v-model.

<br />

```js
<script setup>
const props = defineProps(["modelValue"]);
const emit = defineEmits(["update:modelValue"]);
</script>

<template>
  <input
    v-bind:value="modelValue"
    v-on:input="$emit('update:modelValue', $event.target.value)"
  />
</template>;
```

<br />
Which Vue.js macro can help to simplify the code?

::

::explanation{:type="explanationType"}

**Explanation: The code can be simplified by using the defineModel macro.**

<br />

```js
<script setup>
const model = defineModel()
</script>

<template>
  <input v-model="model" />
</template>
```
