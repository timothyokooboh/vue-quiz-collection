::question{:type="questionType"}
What is the router-link equivalent of this code snippet?

<br />

```js
router.replace({ path: '/home' })
```

::

::explanation{:type="explanationType"}

**Explanation: By setting the replace prop to true, the router-link will replace the current location in the history stack instead of adding a new one.**

<br />

The router.push equivalent is:

```js
router.push({ path: '/home', replace: true })
```

::
