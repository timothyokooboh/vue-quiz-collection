::question{:type="questionType"}
Which built-in Vue.js component is used for caching component instances when dynamically switching between multiple components.

::

::explanation{:type="explanationType"}

**he correct answer is KeepAlive**

<br />

It is used to cache component instances when dynamically switching between multiple component

<br />

```js
<KeepAlive>
  <component :is="activeComponent" />
</KeepAlive>
```

::

::vueTip{:type="tip" :reference="https://vuejs.org/guide/built-ins/keep-alive.html#keepalive" :position="16"}

#text

Wrap dynamic components with KeepAlive to cache their instances and preserve state when switching between them.

#code

```js
<KeepAlive>
  <component :is="activeComponent" />
```
