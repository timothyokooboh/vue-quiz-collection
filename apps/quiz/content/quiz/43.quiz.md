::question{:type="questionType"}
When a user visits /home, how can you redirect them to /?

::

::explanation{:type="explanationType"}

**Explanation: { path: '/home', redirect: '/' } redirects users from /home to /**

<br />

It's important to note that Navigation Guards are not applied on the route that redirects, only on its target. e.g. In the above example, adding a beforeEnter guard to the /home route would not have any effect.
::

::vueTip{:type="tip" :reference="https://router.vuejs.org/guide/essentials/redirect-and-alias.html#Redirect-and-Alias" :position="43"}

#text

Use redirect in route configuration to automatically navigate users from one path to another.

#code

```js
{ path: '/home', redirect: '/' }
```
