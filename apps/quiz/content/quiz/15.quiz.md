::question{:type="questionType"}
Which built-in Vue.js component is used for rendering dynamic components?

::

::explanation{:type="explanationType"}

**Explanation: The correct answer is component**

<br />

It is useful for dynamically switching between components.

<br />

```js
<component :is="tabs[currentTab]"></component>
```
