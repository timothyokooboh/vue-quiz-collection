::question{:type="questionType"}
The code snippet below registers the router instance as a plugin on the Vue application. Which of these is not a feature of the router plugin?

```js
const app = createApp(App)
app.use(router)
app.mount('#app')
```

::

::explanation{:type="explanationType"}

**Explanation: The router plugin does not protect Vue app from every form of XSS attacks.**

<br />

- It registers RouterLink & RouterView as global components.
- It registers $router & $route as global properties.
- It enables useRouter and useRoute composables.
