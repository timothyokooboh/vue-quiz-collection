::question{:type="questionType"}
Attributes not explicitly defined in a component's props or emits can be accessed via which object?

::

::explanation{:type="explanationType"}

**Explanation: Attributes not explicitly defined in a component's props or emits can be accessed via $attrs.**

<br />

$attrs is an object that contains all attributes passed to a component that are not recognized as props.

```js
<script setup>
import { useAttrs } from 'vue'

const attrs = useAttrs()
</script>

<template>
  <div>
    <input v-bind="attrs" />
  </div>
</template>
```

::

::vueTip{:type="tip" :reference="https://vuejs.org/guide/components/attrs.html#fallthrough-attributes" :position="6"}

#text

Use $attrs or useAttrs() to access attributes passed to a component that are not defined as props, enabling fallthrough attribute inheritance.
