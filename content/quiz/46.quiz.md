::question{:type="questionType"}
Which field in the route configuration can be used to pass arbitrary information such as whether a route should be protected by authentication or which transition name should be used?

::

::explanation{:type="explanationType"}

**meta field in the route configuration can be used to pass arbitrary information such as whether a route should be protected by authentication or which transition name should be used.**

<br />


```js
const routes = [
  {
    path: '/posts',
    component: PostsLayout,
    children: [
      {
        path: 'new',
        component: PostsNew,
        // only authenticated users can create posts
        meta: { requiresAuth: true },
      },
      {
        path: ':id',
        component: PostsDetail,
        // anybody can read a post
        meta: { requiresAuth: false },
      },
    ],
  },
]
```

<br />

```js
// User.vue
<script setup>
defineProps({
  id: String
})
</script>

<template>
  <div>
    User {{ id }}
  </div>
</template>
```

```js
  component: PostsLayout,
  children: [
    {
      path: 'new',
      component: PostsNew,
      meta: { requiresAuth: true },
    },
  ],
}
```
