::question{:type="questionType"}
The code snippet below applies a fade transition when navigating between routes. But when navigating from /users/1 to /users/2, no transition was applied. How can you fix this?

<br />

```js
<router-view v-slot="{ Component }">
  <transition name="fade">
    <component :is="Component" />
  </transition>
</router-view>
```

::

::explanation{:type="explanationType"}

**Add a unique key attribute to component to force transition because Vue might automatically reuse components that look alike, thereby avoiding any transition.**

<br />


```js
<router-view v-slot="{ Component, route }">
  <transition name="fade">
    <component :is="Component" :key="route.path" />
  </transition>
</router-view>
```
::

::vueTip{:type="tip" :reference="https://router.vuejs.org/guide/advanced/transitions.html#Forcing-a-transition-between-reused-views" :position="47"}

#text

Add a unique key based on the route to force transitions when Vue reuses components for different routes.

```js
<router-view v-slot="{ Component, route }">

  </transition>
```
