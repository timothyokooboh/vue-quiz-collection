::question{:type="questionType"}
In option stores, which method lets you reset state to its value?

::

::explanation{:type="explanationType"}

**$reset lets you reset the state to its initial value.**

<br />

```js
<script setup>
const store = useStore()

store.$reset()
</script>
```
<br />

In Setup Stores, you need to create your own $reset() method: <br />

```js
export const useCounterStore = defineStore('counter', () => {
  const count = ref(0)

  function $reset() {
    count.value = 0
  }

  return { count, $reset }
})
```
