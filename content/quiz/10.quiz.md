::question{:type="questionType"}
Given the type-based prop declaration below, what will be the equivalent compiled runtime prop declaration?

<br />

```js
defineProps<{ msg: string }>
```

::

::explanation{:type="explanationType"}

**The equivalent compiled runtime prop declaration will be: { msg: { type: String, required: true } }**

<br />

When using type-based prop declarations, Vue will try its best to compile the type annotations into equivalent runtime prop declarations.

::

::vueTip{:type="tip" :reference="https://vuejs.org/guide/components/props.html#prop-validation" :position="10"}

#text

Type-based prop declarations are compiled into runtime prop validations, inferring type and required status from TypeScript types.

#code

```js
defineProps<{ msg: string }>
// Compiles to: { msg: { type: String, required: true } }
```
