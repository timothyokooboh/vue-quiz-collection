::question{:type="questionType"}
What is the purpose of the first argument passed to `defineStore` when creating a store with Pinia?

<br />

```js
import { defineStore } from 'pinia'

export const useAlertsStore = defineStore('alerts', {
  // other options...
})
```

::

::explanation{:type="explanationType"}

**The first argument is used by Pinia to connect the store to the devtools.**

<br />
It is expected to be a unique id of the store across your application

::

::vueTip{:type="tip" :reference="https://pinia.vuejs.org/core-concepts/#Defining-a-Store" :position="48"}

#text

The first argument to defineStore is a unique string id used by Pinia for devtools connection and store identification.

#code

```js
export const useAlertsStore = defineStore('alerts', {
  // options
})
```
