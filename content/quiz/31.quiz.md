::question{:type="questionType"}
What is the built-in component provided by Vue router for creating links to pages?

::

::explanation{:type="explanationType"}

**RouterLink is a built-in component provided by Vue router for creating links to pages.**

<br />

```js
<nav>
  <RouterLink to="/">Go to Home</RouterLink>
  <RouterLink to="/about">Go to About</RouterLink>
</nav>
```

::

::vueTip{:type="tip" :reference="https://router.vuejs.org/api/variables/RouterLink.html" :position="31"}

#text

Use RouterLink to create navigation links that update the URL without reloading the page.

#code

```js
<nav>
  <RouterLink to="/">Go to Home</RouterLink>
  <RouterLink to="/about">Go to About</RouterLink>
```
