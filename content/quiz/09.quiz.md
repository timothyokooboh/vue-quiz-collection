::question{:type="questionType"}
Which of these is not a valid option of defineAsyncComponent?

::

::explanation{:type="explanationType"}

**All the options are valid options of defineAsyncComponent except fallback.**

<br />

**loader** is a function that returns the component to render. **errorComponent** is a component to display if loading the component fails. **timeout** is the time in milliseconds to wait before showing the error component. Other options include **loadingComponent**, which is a component to use while the async component is loading. And **delay**, which is the time in milliseconds to wait before showing the loading component.

<br />

```js
<script setup>
import { defineAsyncComponent } from 'vue'

const AsyncComponent = defineAsyncComponent({
  loader: () => import('./MyComponent.vue'),
  loadingComponent: () => import('./LoadingComponent.vue'),
  errorComponent: () => import('./ErrorComponent.vue'),
  delay: 200,
  timeout: 3000
});
</script>

<template>
  <AsyncComponent v-if="isComponentVisible" />
</template>
```
