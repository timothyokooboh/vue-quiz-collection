::question{:type="questionType"}
Which Vue.js reactivity API takes an object (reactive or plain) or a ref and returns a readonly proxy to the original?
::

::explanation{:type="explanationType"}

**The readonly API in Vue.js takes an object (reactive or plain) or a ref and returns a readonly proxy to the original. It applies deep immutability, that is, any nested property accessed will be readonly as well.**

<br>

```js
const user = ref({ age: 20, email: "test@gmail.com" });
const userCopy = readonly(user);

// mutating the copy will fail and result in a warning!
userCopy.value = { age: 30, email: "test@gmail.com" };

// mutating nested properties will also fail!
userCopy.value.age = 30;
```

::

::vueTip{:type="tip" :reference="https://vuejs.org/api/reactivity-core.html#readonly" :position="4"}

#text

Use the readonly API to create a deep immutable proxy of reactive data, preventing mutations to the original object and its nested properties.

#code

```js
const user = ref({ age: 20, email: "test@gmail.com" });
const userCopy = readonly(user);

// Mutations will fail with a warning
userCopy.value.age = 30;
```
